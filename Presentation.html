4 COMPOSITE FEATURES

Engineering Predictive Signal

01 - BINARY FLAG
pdays_was_contacted
pdays ≠ 999 → 1
Decouples the 999 sentinel from the numeric scale. pdays=999 means "never contacted" — a categorical distinction, not a measurement.
02 - RFM-INSPIRED
engagement_score
(previous+1) / (campaign+1)
How engaged is the client relative to contact frequency? Higher = proven contact history with fewer touches needed.
03 - INTERACTION
loan_burden
housing_bin + loan_bin
Total active loans (0, 1, or 2). Captures the combined financial pressure from housing + personal loans.
04 - MACRO PROXY
euribor_cpi_spread
euribor3m − CPI/100
Real interest rate proxy. Positive spread = term deposits more economically attractive relative to inflation.
ALL 4 IN TOP-20 Each engineered feature appears in top-20 importance rankings — confirming domain-driven feature engineering adds real signal.

CONFUSION MATRIX · METRICS FORMULAS

How We Evaluated the Models

Confusion Matrix — foundation of classification evaluation. We evaluate on the original imbalanced test set (8,238 samples) to reflect real-world deployment.
TRUE NEGATIVE
TN
Actual No → Pred No ✓
FALSE POSITIVE
FP
Actual No → Pred Yes ✗
FALSE NEGATIVE
FN
Actual Yes → Pred No ✗
TRUE POSITIVE
TP
Actual Yes → Pred Yes ✓
METRICS FORMULAS
Precision = TP / (TP + FP)     → minimize wasted outreach
Recall    = TP / (TP + FN)     → find more subscribers
F1        = 2·P·R / (P + R)    → harmonic mean of P & R
ROC-AUC   = area under ROC     → ranking quality
Accuracy  = (TP+TN) / Total    → DECEPTIVE for imbalanced data!

DATA LEAKAGE WARNING
19 / 23
WHY WE EXCLUDED THE STRONGEST PREDICTOR

The Duration Trap

⚠ Duration is a post-hoc feature
You only know the call length after the call ends. Including it means predicting the past — the model knows the outcome before making the prediction. This is the textbook definition of data leakage.
0.952
AUC WITH duration
0.798
AUC WITHOUT duration
The gap 0.952 → 0.798 (−0.154 AUC) quantifies the cost of realism. The duration-inclusive model is a theoretical upper bound — useful as a sanity check that our pipeline works, but never deployable. Our production XGBoost achieves 0.798 AUC without any leakage. 
# Production features — duration EXCLUDED
production_num_cols = ['age','campaign','pdays','previous'] + econ_cols
# Reference only — duration INCLUDED (DO NOT DEPLOY)
full_num_cols = production_num_cols + ['duration']

CHALLENGES & LESSONS
21 / 23
WHAT WE LEARNED

Challenges & Solutions

CORE LESSON
The best model deploys — not the one with the highest AUC.
Removing duration drops AUC from 0.95 to 0.80 — but makes the model production-viable.
01
Class Imbalance (11.3%)
Accuracy is deceptive at 88.7%. SMOTE + F1/Precision/Recall metrics + lift analysis.
02
Data Leakage (Duration)
Strongest predictor is post-hoc. Two models: production (AUC 0.80) vs reference-with-leakage (AUC 0.95).
03
Economic Multicollinearity
euribor3m ↔ nr.employed: ρ = 0.94. Domain shift risk: 2008–2010 crisis data.
04
Double Correction Trap
SMOTE + class_weight over-corrects. Single-method balancing yields better results.
